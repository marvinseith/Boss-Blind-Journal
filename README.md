# Boss Blind Journal

A simple webapp to track Balatro boss blinds during your runs.

## Features

- **Complete Boss List** - All 28 boss blinds with names and descriptions
- **Quick Search** - Find bosses by name or effect
- **Mark as Seen** - Click any boss to mark it as encountered
- **Filter View** - Toggle between all bosses and only encountered ones
- **Progress Tracking** - See how many bosses you've faced
- **New Run Reset** - Clear all progress when starting fresh

## Usage

1. Open the webapp in your browser
2. Click on boss blinds as you encounter them during your run
3. Use the "Encountered" filter to quickly review which bosses you've faced
4. Click "New Run" to reset when starting a new game

## Why?

Perfect for players who want to keep track of which boss blinds they've already encountered during a run without having to remember or scroll through their history.

## Tech Stack

Built with React and Tailwind CSS as a standalone single-page application.
